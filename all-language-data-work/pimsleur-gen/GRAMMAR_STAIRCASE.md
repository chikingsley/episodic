# Pimsleur Grammar Staircase: All 5 Levels

## How to Read This Document

This is the formal grammar progression across all 5 Pimsleur French levels (150 lessons, A0→B2).

Each grammar structure is listed with:
- **Introduced**: First level/lesson where the structure appears
- **Drilled**: Where it gets intensive repetition
- **Mastered**: Where the learner uses it without English prompting (automatic)
- **CEFR**: Standard level where this structure is expected

## 7. Key "Handle" Constructions

These are the high-frequency patterns that act as scaffolding for everything else:

| Construction | Level | Function |
|-------------|-------|----------|
| je voudrais + infinitive | L1 | Polite requests ("I would like to...") |
| est-ce que + statement | L1 | Question formation |
| je vais + infinitive | L1 | Near future ("I'm going to...") |
| il faut + infinitive | L2 | Impersonal necessity ("one must...") |
| il faut que + subjunctive | L3-L4 | Personal necessity ("I must...") |
| si + imparfait → conditionnel | L4 | Hypotheticals ("If I had... I would...") |
| j'aurais + past participle | L4-L5 | Past regrets ("I would have...") |

## Cross-Reference: Where Pimsleur Differs from Speak and Duolingo

### Things Pimsleur Teaches FIRST That Others Don't

| Structure | Pimsleur | Speak | Duolingo |
|-----------|---------|-------|----------|
| Polite conditional (voudrais) | L1-L3 (very early) | L1 unit 1 (aimerais) | Section 2 |
| Inversion questions (est-il?) | L1-L7 | Not emphasized | Section 3 |
| Est-ce que as primary question form | L1-L1 | L1 unit 1 | Section 1 |
| Partitive articles (du, de la) | L1-L7 | Elementary 2 | Section 2 |

### Things Pimsleur Teaches LATE That Others Teach Early

| Structure | Pimsleur | Speak | Notes |
|-----------|---------|-------|-------|
| Tu/vous distinction | L1-L28 | Beginner 1 | Pimsleur uses vous-only for 27 lessons |
| aimer (to like) | L1-L29 | Beginner 1 Unit 3 | Speak teaches personal expression early |
| Informal greetings (salut) | L2-L9 | Beginner 1 Unit 1 | Pimsleur is formal-first |
| chercher (to look for) | Never | Beginner 1 Unit 2 | Pimsleur uses "où est..." instead |
| Payment vocabulary (par carte) | Never | Beginner 1 Unit 4 | Pimsleur uses outdated francs |
| J'apprends le français | L3 | Beginner 1 Unit 3 | Learner-specific phrase |

### Things Missing from Pimsleur Entirely

Based on the A0-A1 gap analysis, Pimsleur never teaches:
- `avoir faim/soif` (to be hungry/thirsty)
- `chercher` (to look for)
- `par carte / en espèces` (payment methods)
- `être en train de` (ongoing action)
- `avoir envie de` (to feel like doing)
- `être d'accord` (to agree)
- Modern currency (euros) -- uses francs
- Most reflexive verbs in Level 1 (se lever, se coucher, s'habiller, etc.)
- Ordinal numbers (premier, deuxième...)

## Implications for Course Generation

### The Grammar Staircase is Language-Specific But the Pattern is Universal

The specific structures (subjunctive, imparfait, etc.) are French-specific. But the PATTERN of introduction is universal:

1. **Survival forms first** (present tense, basic questions, basic negation)
2. **Polite forms early** (conditional for requests)
3. **Past tenses in the middle** (when you can already speak in the present)
4. **Complex moods late** (subjunctive, conditional perfect)
5. **Nuance last** (register, reported speech, advanced conditionals)

For a new language, you'd follow the same staircase but with that language's grammar:
- Japanese: polite forms (です/ます) → plain forms → て-form → potential → conditional → passive
- Spanish: present → preterite/imperfect → subjunctive → conditional perfect
- German: present → Perfekt → Präteritum → Konjunktiv II → Konjunktiv I

### Handle Constructions are the Key to Curriculum Design

The "handle constructions" (je voudrais +, est-ce que +, je vais +, il faut que +) are the real curriculum. Grammar structures exist to serve these handles. When designing a new course:

1. Identify the 6-8 handle constructions for the target language
2. Order them by survival utility
3. Map the grammar structures needed to support each handle
4. That gives you the grammar staircase
